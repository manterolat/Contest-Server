Plug-n-play

Very easy to deploy. Add your configuration and launch.

# Easy to deploy

# Download the code
git clone https://github.com/FSU-ACM/Contest-Server.git

# Copy in your config
cp /your/docker/.env ./.env
cp /your/flask/production.py ./config/production.py

# Build the images
docker-compose build

# Deploy the suite
docker-compose up -d
